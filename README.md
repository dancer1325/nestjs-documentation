## how to build?

* -- via -- [Dgeni documentation generator](https://github.com/angular/dgeni)
* `npm run build`
  * build the project | `dist/`
* `npm run build:watch`
  * content changes -- will be -- recompiled & rebuilt
* `npm run build:prod`
  * production build
